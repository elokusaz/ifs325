Scanning the 16-pixel grid reveals the following symbol counts:

Step 2: Build Huffman Tree
The tree is built by repeatedly combining the two nodes with the lowest frequency.
  1. Combine K(1) + W(4) to create an internal node with weight 5.
  2. Combine this Internal Node(5) + B(11) to create the final root node with weight 16.

Huffman Tree Structure

(Root, 16) / \ '0' '1' (Node, 5) (Blue, 11) / \ '0' '1' (Black, 1) (White, 4)
Step 3: Assign Binary Codes
Traversing the tree from the root (convention: left=0, right=1):
  • Blue (B): Right → 1 (1 bit)
  • White (W): Left, Right → 01 (2 bits)
  • Black (K): Left, Left → 00 (2 bits)
